HTML (HyperText Markup Language)
Язык разметки, используемый для создания и структурирования контента на веб-странице.
CSS (Cascading Style Sheets)
Язык стилей, используемый для определения внешнего вида веб-страницы.
JavaScript
Язык программирования, которая позволяет создавать динамически обновляемый контент, управлять мультимедиа, анимировать изображения и многое другое.
